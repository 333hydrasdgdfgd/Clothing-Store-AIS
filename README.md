# Clothing Store AIS (AURA OS)

AURA OS is a complete apparel operations system designed for boutique owners, inventory managers, sales staff, and accounting professionals. It provides an all-in-one platform for managing retail operations efficiently.

---

## Overview

AURA OS was developed based on real user feedback to simplify and streamline clothing store management.  
It supports role-based access, sales tracking, inventory control, customer management, and full accounting functions.

---

## Key Features

- Detailed sales and purchase analytics via dashboard and export  
- Role-based access control (Sales Manager, Purchasing Staff, Inventory Clerk, Cashier, Accounting)  
- Inventory tracking with audit trails for accuracy and compliance  
- Mobile-responsive design optimized for tablets and POS devices  
- Quick-edit and zoom features for fast product and catalog management  
- Integrated modules for suppliers, customers, employees, and shipping  
- Full accounting system (invoices, receipts, expenses, profit & loss reporting)
